**Project Overview**

This project analyzes hospitality data to generate insights on hotel performance, customer behavior, and revenue trends. It supports data-driven decision-making through interactive dashboards.

** Objectives**

Analyze revenue and booking trends

Monitor occupancy and cancellation rates

Identify peak booking periods

Improve business decisions using data insights

 **Tools & Technologies**

Excel (Data Cleaning)

SQL (Data Extraction & Transformation)

Power BI (Dashboard Development)

Tableau (Data Visualization)

** Key Metrics**

Total Revenue

Occupancy Rate

Cancellation Rate

Booking Trends

Customer Segmentation

** Key Insights**

Higher occupancy observed during weekends

Cancellation rate impacts revenue significantly

Premium room categories generate more revenue
